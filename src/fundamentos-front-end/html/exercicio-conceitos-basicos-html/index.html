Introdução ao HTML

O que é HTML ?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar o conteúdo de páginas na web. É a base de qualquer site ou aplicação web, funcionando como um conjunto de elementos e tags que descrevem e organizam os diferentes tipos de conteúdo, como texto, imagens, links, vídeos e tabelas.

A tag <b> em HTML é usada para destacar um texto ou uma parte de texto visualmente em negrito (bold). Contudo, o significado semântico dela é mais limitado.

A tag <strong> em HTML é usada para indicar que um texto ou uma parte de texto com grande importância ou ênfase. Por padrão, ela aplica o estilo negrito (bold) ao texto, mas seu propósito principal é semântico.

A tag <i> em HTML é usada para representar texto que precisa ser destacado de maneira diferente, geralmente exibido em itálico. No entanto, sua principal função é visual e não tem semântica forte associada.

A tag <em> em HTML é usada para indicar que um trecho de texto a ser enfatizado. O significado da ênfase é semântico, ou seja, ela comunica que o conteúdo destacado tem maior relevância ou importância no contexto. Visualmente, o texto é exibido em itálico por padrão, mas o estilo pode ser personalizado com CSS.

A tag <mark> em HTML é usada para destacar texto importante ou relevante, aplicando um estilo visual padrão de fundo amarelo (como se fosse "marcado com um marcador de texto"). Ela é útil para chamar a atenção do usuário para informações específicas.

A tag <del> em HTML é usada para indicar texto que foi excluído ou removido de um documento. Ela fornece significado semântico, indicando que o texto foi deliberadamente marcado como removido, geralmente porque foi substituído ou atualizado.

A tag <sup>; em HTML é usada para representar texto em formato sobrescrito (ou índice superior). O texto aparece ligeiramente acima da linha base do texto normal e com tamanho reduzido.
